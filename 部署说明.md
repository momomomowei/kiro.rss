# Docker 部署

1. 准备配置：`mkdir -p config`，把 `config.example.json` 复制为 `config/config.json`，把凭据保存为 `config/credentials.json`。
2. 构建本地镜像：`docker compose -f docker-compose.local.yml build`
3. 运行本地镜像：`docker compose -f docker-compose.local-run.yml up -d --force-recreate`
4. 本地访问：`http://127.0.0.1:9990/admin`
5. 配置挂载：`./config/ -> /app/config/`
6. 域名/反代访问：把 `config/config.json` 里的 `"host": "127.0.0.1"` 改成 `"host": "0.0.0.0"`。
7. 停止服务：`docker compose -f docker-compose.local-run.yml down`
